# JiraGuard AI

JiraGuard AI is an AI-powered workflow assistant that helps support and product teams turn unclear support requests into structured Jira tickets.

## Why I Built This

In support and engineering workflows, unclear tickets can slow teams down. Important details often get lost, tickets are missing context, and engineers may need to ask follow-up questions before work can begin.

I built JiraGuard AI to help teams create better handoffs from the start.

## Features

- Converts support request text into structured Jira ticket content
- Generates summaries, reproduction steps, expected results, and impact analysis
- Uses schema-driven validation for consistent output
- Designed around real Jira and support workflows

## Tech Stack

- TypeScript
- Node.js
- AI workflow logic
- Schema validation

## Current Status

This project is currently in active development and has been used locally to process and refine support tickets while improving ticket clarity and engineering handoff quality.
